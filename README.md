# Household Inventory Management System

## Project Overview
The Household Inventory Management System is a web-based application designed to help users manage and track household items efficiently. It allows users to monitor inventory levels, track usage history, receive notifications, and maintain secure access through authentication.

## Tech Stack
- **Backend:** Java, Spring Boot
- **Frontend:** Thymeleaf, HTML, CSS
- **Database:** H2 (in-memory) / MySQL
- **Security:** Spring Security, BCrypt password encryption
- **Build Tool:** Maven

## Features
- User Registration and Login with secure authentication
- Add, update, and delete household inventory items
- Track item quantities and low-stock thresholds
- Usage history tracking for consumed items
- Email notifications for low stock and important alerts
- Role-based secure access using Spring Security

## How to Run the Project
1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Configure database (H2 or MySQL)
4. Run the Spring Boot application
5. Access the application in the browser
